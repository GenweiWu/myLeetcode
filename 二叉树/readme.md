## 二叉树



### 1. 遍历二叉树

> 先中后说的根节点，左右永远都是先左后右  
> 先按照父层节点处理，碰到孩子就先处理孩子节点

- 先序遍历（根->左->右）
- 中序遍历   (左->根->右)
- 后序遍历   (左->右->根)
- 层次遍历：只需按层次遍历即可
