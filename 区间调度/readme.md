## 总体思路
- 区间问题，先排序，然后分情况：能合并、不能合并分别进行处理即可

- 并集问题好处理，所有区间平铺(比如多个非重叠区间分组可以合并为统一处理)也可以
- 交集问题麻烦一些，不要平铺(一般针对两个非重叠区分组，分别处理)
