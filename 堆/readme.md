
## 基础概念

- 堆就是用数组实现的二叉树

- 大顶堆、小顶堆
  - 大顶堆：
    ```
    - 每个节点都大于等于它的两个子节点
    - 顶部是最大值
    ```
  - 小顶堆：
    ```
    - 每个节点都小于等于它的两个子节点
    - 顶部是最小值
    ```


- 优先队列
