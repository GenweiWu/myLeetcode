

## 方法1
> 重点理解：位操作的小技巧   
> https://leetcode-cn.com/problems/number-of-1-bits/solution/wei-1de-ge-shu-by-leetcode/
```java
class Solution {
      public int[] countBits(int num) {

        int[] result = new int[num + 1];
        for (int i = 0; i <= num; i++) {
            result[i] = doCountBits(i);
        }
        return result;
    }

    private int doCountBits(int num) {
        if (num == 0) {
            return 0;
        }

        int count = 0;
        while (num != 0) {
            num = num & (num - 1);
            count++;
        }
        return count;
    }
}
```
