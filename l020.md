有效的括号
</summary>

> https://leetcode-cn.com/problems/valid-parentheses/
```
给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，判断字符串是否有效。

有效字符串需满足：

左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。
注意空字符串可被认为是有效字符串。
```

</details>

### 总结

#### 1)方法1：
```
括号匹配是栈的经典用法，匹配到左括号就push，匹配到右括号就对pop出来的元素进行匹配判断，一旦不匹配则失败；
最后如果判断的时候栈已经为空了'比如{}}'，或者最最后栈不为空'比如{{}'都算失败了
```

#### 2)方法2：
```
优化下，不用栈行不行，抽象成正负数
```
